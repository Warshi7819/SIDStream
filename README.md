those retro SID music files! SID? Music file format commanly known to house Commodore 64 music!

## Overview

SIDStreamer lets you load and play SID music files. It provides a simple UI for playback control.

## Features

- Load and play SID files (.sid)
- Playback controls: Play, Stop

## Requirements

- .NET 10
- Windows 10 or later
- Visual Studio 2026 (recommended) or compatible IDE
- MonoGame Framework
