Remember the 90's? Finally a real shaped Windows application is Back! This beauty will let you stream 
those retro SID music files! 

What is SID? A music file format (of sorts) commonly known to house Commodore 64 music!

## Overview

SIDStreamer lets you load and play SID music files! It doesn't get more retro than that. 

## Features

- Load and play SID files (.sid)
- Playback controls: Play, Stop

## Requirements

- .NET 10
- Windows 10 or later
- Visual Studio 2026 (recommended) or compatible IDE
- MonoGame Framework
